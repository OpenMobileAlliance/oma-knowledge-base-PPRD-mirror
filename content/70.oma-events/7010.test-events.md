## OMA LwM2M Test Event (SVE-44)

The Open Mobile Alliance (OMA) is pleased to announce the opening of registration for the **OMA LwM2M Test Event (SVE-44)**, taking place from **20 to 23 April 2026** in **Krakow, Poland**, and hosted by **AVSystem**.

This event brings together implementers of **OMA Lightweight M2M (LwM2M)** Clients and Servers for confidential, multi-vendor interoperability testing in a structured and collaborative environment.

### About OMA Test Events

OMA Test Event is a confidential and secure, multi-vendor interoperability testing event. Participants use Client and/or Server implementations of OMA protocols (<a href="https://www.openmobilealliance.org/release/LightweightM2M/" target="_blank">LwM2M v1.0, v1.1, and v1.2</a> for this event) for structured cross-vendor testing.

OMA will provide official <a href="https://www.openmobilealliance.org/release/LightweightM2M/ETS/" target="_blank">test cases</a> and publish aggregate (non-attributed) test results. Participants may optionally publish their individual implementation test results on the <a href="https://www.openmobilealliance.org/specifications/resources/product-listing" target="_blank">OMA Product Listing site</a>.

::ShAlert
üîí **Non-Disclosure Agreement (NDA) Required**
Participation in OMA Test Event requires a valid **Non-Disclosure Agreement (NDA)** to ensure a confidential and secure multi-vendor testing environment.  
If you do not already have an NDA on file with OMA, please review and sign the NDA using the link below before participating in the event:
